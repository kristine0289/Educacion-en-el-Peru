¿Cuál es el proposito de esta página?

EL proposito de esta página es exponer y denunciar la realidad que muchas veces se presenta dentro de instituciones educativas gratuitas peruanas, tanto dentro de nivel inicial como primaria y secundaria, asi como posibles soluciones o contribuciones que podemos realizar para resolver esta problemática.
